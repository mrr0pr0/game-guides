What Guide Types to Use
Based on the IGN Assassin's Creed Odyssey wiki, here are the standard guide categories you should use:

1. walkthrough
Use for story-related content and quest guides.

Examples:

Main Story Walkthrough
Side Quests Guide
Chapter-by-Chapter Guides
Mission Walkthroughs
Choices and Consequences
Ending Guide
Typical Content:

Step-by-step instructions
Quest objectives
Boss strategies
Story choices and their outcomes
2. collectibles
Use for items, secrets, and things to find in the game.

Examples:

All Collectible Locations
Hidden Items Guide
Treasure Map Locations
Secret Area Locations
Easter Eggs and References
Achievement/Trophy Guide
Cultist/Enemy Locations
Typical Content:

Item locations with coordinates
Maps and screenshots
Rewards for collecting
Tips for finding hard-to-reach items
3. tips
Use for general advice, strategies, and character guides.

Examples:

Best practices

How-to explanations

Example Slug Generation

Guide Title	Slug

All Collectibles Locations	all-collectibles-locations
50 Essential Tips and Tricks	essential-tips-and-tricks

Romance Options - All Relationships	romance-options-guide
Formula:

Convert to lowercase
Replace spaces with hyphens
Remove special characters (- : , . !)
Keep letters, numbers, and hyphens only

Pro Tips
Be Consistent: Use the same 3 categories (walkthrough, collectibles, tips)
Balance: Try to have guides in all 3 categories
Priority: Start with walkthrough and tips, add collectibles later
Updates: You can always add more guides to existing games
Flexibility: If a guide fits multiple categories, choose the most relevant one
Remember: The guide_type field creates the sidebar categories, so choose wisely!
